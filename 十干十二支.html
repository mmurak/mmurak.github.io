十干十二支覚え書き

　このページでは、日付（年、月、日）の十干十二支表現についてまとめている。（十二支はこの他にも、方角や時間を表すために用いられている。）

　十干十二支と日付を相互変換する上で、十干、十二支を数値化する必要がある。このページでは一律、以下の数値化テーブルを使用している（各セクションでは便宜上、同じものを適宜引用している）。

十干
1234567890
甲乙丙丁戊己庚辛壬癸
十二支
12345678910110
子丑寅卯辰巳午未申酉戌亥

年の干支から西暦年を求める

  1. 十干を数値に変換し、その値をAと置く。
    1234567890
    甲乙丙丁戊己庚辛壬癸
  2. 十二支を数値に変換し、その値をBと置く。
    12345678910110
    子丑寅卯辰巳午未申酉戌亥
  3. AからBを引き、結果を2で割る。
  4. 値が負の場合には6を足す。
  5. その値の右横に、Aの値を付加する（つまり10倍してAを加算する）。
  6. この値が十干十二支における甲子からの通番となる。
  7. 通番に3を加えた後、60の倍数（年の干支は60年周期で繰り返されるため）を加える。

月の干支から西暦年下1桁と月（旧暦）を求める

  1. 十二支を数値に変換して、Aと置く。
    12345678910110
    子丑寅卯辰巳午未申酉戌亥
  2. Aの値に10を足し、12を超えたら12を引く（つまり12で割ったあまりを求める）。
    これが旧暦の月となる。
  3. 十干を数値に変換して、Bと置く。
    1234567890
    甲乙丙丁戊己庚辛壬癸
  4. Bの値から、上で求めた「月の数値の下1桁」を引き、結果を2で割る。
  5. 値が負の場合には10を足す。
  6. その結果に、3あるいは8を足した値の下1桁が年の下1桁となる（月の干支は5年で1周するため、年の下1桁は2種類考えられる）。

西暦年からその年の十干を求める

  1. 西暦年から3を引く。
  2. 下1桁が十干を表す。
    1234567890
    甲乙丙丁戊己庚辛壬癸

西暦年からその年の十二支を求める

  1. 西暦年を3で割ったあまりをAと置く。これには以下の手順を実行する。
    1. 西暦年の各桁を加算する。
      例：1978年の場合、1 + 9 + 7 + 8 = 25
      （なお、さらに各桁を加算することも可能）
    2. 合計を3で割ったあまりをAとする。
      例：25 % 3 = 1
  2. 西暦年+1を4で割ったあまりをBと置く。これには以下の手順を実行する。
    • 西暦年が偶数の場合、西暦年の下2桁を2度2で割った値が整数かどうかを見極める。
      　（整数であればあまりは 1、小数点以下があればあまりは 3）
      　　例：1978年の場合、（1回目）78 / 2 = 39、（2回目）39 / 2 = 19.5、、、つまりあまりは3となる。
    • 西暦年が奇数の場合、西暦年の下2桁に1を足した後、2度2で割った値が整数かどうかを見極める。
      　（整数であればあまりは 0、小数点以下があればあまりは 2）
      　　例：1965年の場合、（1回目）66 / 2 = 33、（2回目）33 / 2 = 16.5、、、つまりあまりは2となる。
  3. Aの値からBの値を引き、Cと置く。ただし、負になった場合は3を加算する。
    例：1 - 3 + 3 = 1
  4. Cを4倍し、Bの値を加算した値が十二支を表す。
    例：1 × 4 + 3
    12345678910110
    子丑寅卯辰巳午未申酉戌亥

年と月（旧暦）からその月の干支を求める

  1. 年から3を引き、結果の下1桁を取り出す。
  2. その値が5以上であれば、さらに5を引く。
  3. 結果の値（つまり0〜4の値）を2倍して、月の数字を足す。
  4. 10のあまりを求めると、その値が十干となる。
    1234567890
    甲乙丙丁戊己庚辛壬癸
  5. 月の数字に2を足し、12で割ったあまりを求めると、その値が十二支となる。
    12345678910110
    子丑寅卯辰巳午未申酉戌亥

グレゴリオ暦による年月日からその日の日干支を求める

  1. 年の下2桁を20倍した後、元の下2桁を加え、その値を2度、2で割った値をAと置く（小数点以下は切り捨てる）。
    （結果的に5.25倍したことになる）。
  2. 1900年〜1999年であれば10を、2000年〜2099年であれば55をBと置く（さらに古い日付を求める場合には、以下の説明を参照）。
  3. 以下で示した月の値をCと置く（ただし「++」と表記された小の月は、左隣の値に31を足した値をCとする）。
    1月2月3月4月5月6月7月8月9月10月11月12月
    0++59++0++132++33++34
  4. 日の値をDとする。ただし、閏年の1月または2月は1を減算しておく。
  5. A、B、C、Dの値をすべて足し合わせ、60で割ったあまりをRと置く。
  6. Rの下1桁が十干を表す。
    1234567890
    甲乙丙丁戊己庚辛壬癸
  7. Rから、12、24、36、48のいずれかの値（結果が負にならない最大値）を引いた値が十二支を表す。
    12345678910110
    子丑寅卯辰巳午未申酉戌亥

※ 1899年以前のBの求め方：

　1899年以前のBは、以下の手順で求められる。

  1. 西暦年を100で割った値（西暦年が4桁ならば上2桁）をYと置く。
  2. Yを4で割った値をZと置く（小数点以下は切り捨てる）。
  3. Yを44倍し、Zを足し、さらに10を足した値を60で割ったあまりがBとなる。

　なお、グレゴリオ暦は1582年10月15日から施行されている暦制度であるため、それ以前の日付（おそらくはユリウス暦で表現されている）の日干支を求める際には、その日付をいったん先発グレゴリオ暦に変換する必要がある点に留意されたい。
